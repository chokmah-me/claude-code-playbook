# Extract Function to Module

Surgical extraction of functions into new modules with validation.

## Usage
Use for targeted decomposition of large files.

## Process
1. Identify function to extract
2. Analyze dependencies
3. Create new module
4. Update imports
5. Add tests
6. Validate (type-check, tests, lint)

## Token Cost
~5K tokens per function
