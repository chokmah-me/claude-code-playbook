# Modernize Code Patterns

Update code to use modern TypeScript patterns.

## Updates Include
- `.then()` → `async/await`
- `throw/catch` → `Result<T,E>`
- Classes → Factory functions
- `any` types → Proper types
- Add JSDoc comments

## Usage
Use after extracting code or for standalone pattern updates.

## Token Cost
~4K tokens per file

See SKILL.md and knowledge/typescript-style.md for patterns.
